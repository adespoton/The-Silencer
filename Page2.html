The Silencer Help

Settings

Many of the settings in The Silencer are experimental and some do not work well with each other. For example, reloading the game after adjusting tile size is recommended, as the player and NPCs will not move and could end up stuck.

For ordinary gameplay, any settings in "Experimental Features" should not be changed from their defaults, but player size and maximum text size can be changed without any consequences. Control inversion can make for some interesting and less intuitive gameplay, but can be annoying and so is designed to be used only for people who want to make life difficult for themselves.

The game can also be restarted using the "Restart Game on Next Launch" feature.
